## 你好！ :wave:

[English](https://github.com/NJU-CITE-Compression/.github/blob/main/profile/README.md) | 简体中文

这里是南京大学 [CITE](https://cite.nju.edu.cn)（计算成像技术）实验室的编码组。我们主要进行基于深度学习的视觉压缩算法研究，包括图像、视频、3D和特征压缩。同时我们也在研究将压缩算法转化为芯片设计。
